# 🥁 JavaScript Drum Kit

An interactive keyboard-powered drum kit built with **vanilla JavaScript**.  
Press keys on your keyboard to trigger drum sounds with smooth CSS animations.

## 🚀 Features

- 🎹 Play drum sounds using keyboard keys  
- 🔊 Real-time audio playback  
- 🎨 CSS animations triggered on key press  
- ⏱️ `transitionend` event handling for smooth visual reset  
- ⚡ Built with pure JavaScript (no frameworks)

---

## 🛠️ Built With

- **HTML5**
- **CSS3** (Transitions & Animations)
- **JavaScript** (Key events, Audio API, DOM manipulation)

---

## 🧠 What I Practiced

- Handling `keydown` events  
- Dynamically playing audio with the `<audio>` element  
- Using `data-*` attributes to connect keys to sounds  
- Working with the `transitionend` event  
- Writing clean, event-driven front-end logic
